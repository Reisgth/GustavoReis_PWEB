Com tradutores, ou como são mais conhecidas: linguagens de programação. Hoje, existem muitas (realmente muitas) linguagens, ou seja, muitas formas diferentes de passar informações para a máquina. De forma geral, elas se dividem em:

  • Linguagens de alto nível: São mais recentes, com sintaxe e semântica que se aproximam da linguagem humana Alguns exemplos: Python, Rails, Java, C#, Swift, PHP, JavaScript
  • Linguagens de baixo nível: São mais antigas, com sintaxe e semântica que se aproximam da linguagem da máquina (binária) Alguns exemplos: C, C++

O que é o back-end?

O back-end, como o proprio nome já diz é o que está por tras dos códigos, processamento, entre outras coisas como rotas de acesso, ou 'endpoints'. A linguagem atual mais utilizada para o desenvolvimento back-end é o nodeJs, porém muitas outras linguagens são nativamente desenvolvidas para o back-end como java, por exemplo, que fornece acesso a comandos sql diretamente na linguagem.
